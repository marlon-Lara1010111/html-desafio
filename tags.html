Aqui estão algumas outras Tags do HTML

a - Âncora (link): Usado para criar hiperlinks para outras páginas ou recursos.

img - Imagem: Utilizado para incorporar imagens em uma página web.

ul - Lista não ordenada: Cria uma lista de itens sem uma ordem específica.

ol - Lista ordenada: Cria uma lista de itens com uma ordem numérica.

li - Item de lista: Define um item em uma lista (dentro de ul ou ol).

button - Botão: Cria um botão clicável que pode executar ações no navegador.
